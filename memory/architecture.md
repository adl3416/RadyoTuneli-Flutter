# Architecture

- Flutter app for live Turkish radio listening.
- Main app code lives in `lib/`.
- Platform integrations live in `android/`, `ios/`, `web/`, `linux/`, and `windows/`.
- Static media and bundled resources live in `assets/` and `public/`.
- Tests live in `test/`.
- Root markdown files contain feature notes, Android Auto notes, reports, and historical debugging context.
- Prefer reading focused files near the feature being changed instead of broad project scans.
